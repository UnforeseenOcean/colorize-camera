colorize-camera
===============

Makes sounds according to what the camera captures.

Actually, it analyses just 16 points on the video, evenly spaced. This control points are indicated over the video, to make user interaction easier.

The sounds are triggered and parameterized according to the color variations in these positions. Theres is too a visual feedback indicating the strongest color component of the sector when it changes.

The user can click in any of the 16 sectors of the video to turn their use on/off.

The video on the screen is shown horizontally inverted, so it behaves as the user was looking at a mirror. This helps a lot user interaction (if non-inverted, the experience gets annoyingly non-intuitive)

----

Gera sons de acordo com o que a câmera captura.

No momento, analisa apenas 16 pontos no vídeo, igualmente espaçados. Esses pontos de controle são indicados por sobre o vídeo, para facilitar a interação com o usuário.

Os sons são disparados e parametrizados de acordo com as variações de cor nessas posições. Há também uma resposta visual indicando a mais forte componente de cor no setor quando ela muda.

O usuário pode clicar em qualquer um dos setores para ligar/desligar seu uso.

O vídeo na tela é mostrado invertido horizontalmente, como se a tela do computador fosse um espelho. Isso facilita imensamente a experiência do usuário (se não invertido o vídeo, o uso se torna irritantemente não-intuitivo).
